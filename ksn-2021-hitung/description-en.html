Description

Pak Dengklek is a ball manufacturer. To aid his work, he has ordered a machine that can count the number of balls he makes in a day. With it, Pak Dengklek only needs to put in all the balls he made, then the machine will count and show an integer representing the number of balls that have entered the machine.

After receiving the machine, Pak Dengklek heard a rumor that the machine sometimes count balls incorrectly. In order to be able to make a complaint if the machine has the problem, Pak Dengklek will record a video showing the performance of the machine received.

In the video, Pak Dengklek plans to put $Q$ balls into the machine one by one and check whether the integer shown matches the number of balls that have entered the machine. In detail, Pak Dengklek will do the following.

  1. Define $x$ as the number of balls that have entered the machine. Initially, $x$ is $0$.
  2. Repeat the following steps $Q$ times.
    1. Put one ball into the machine and the value of $x$ is changed into $x + 1$.
    2. If the machine shows the correct integer, which is $x$, continue this process.
    3. If not, remove all balls from the machine, and the value of $x$ is reset to $0$.

During the recording, it is possible that the machine will always show correct integers. It is also possible that the machine will always show incorrect integers.

Because Pak Dengklek is busy, he asks for your help to record the video. Help Pak Dengklek determine whether the machine's calculation is correct every time a ball entered the machine!

Constraints

  • $1 \leq Q \leq 10\;000$

Subtasks

  1. (10 points) It is guaranteed that the machine always show correct integers.
  2. (10 points) It is guaranteed that the machine always show incorrect integers.
  3. (15 points) It is guaranteed that the machine always show integer $1$.
  4. (65 points) No additional constraints.

Problem Type Information

This problem is an "interactive" problem. In this problem, you will interact with the judge's program through standard input (stdin) and standard output (stdout). See the following interaction carefully.

Interaction

Initially, you will be given an integer $Q$ in the following format: 

Q

Then for $Q$ times, you will put in one ball and be given an integer $B$ in the following format: 

B

This means that the machine shows integer $B$. It is guaranteed that $B$ fits in signed $32$-bit integer. For each of the inputs, output a string $S$ with value BENAR (means correct) if the machine shows the correct integer, and SALAH (means incorrect) otherwise. Output in the following format: 

S

If you output $S$ with the incorrect value, then you will receive Wrong Answer for the test case. You will receive Accepted verdict if the value of every $S$ you output is correct.

Warning

Always flush every time your program gives an output.

  • For library stdio.h (cstdio):
    fflush(stdout);
  • For library iostream:
    std::cout << std::flush;
