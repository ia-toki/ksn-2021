Description

There are $N$ balls numbered from $1$ to $N$. These balls look identical but may be made from different materials. For easier identification, Pak Dengklek will colour these balls such that balls with different materials have different colours, and balls with the same material have the same colour.

To differentiate between one ball and the others, Pak Dengklek uses a device named Ballanyzer. However, this device can only be used at most $Q$ times. In one usage, Pak Dengklek chooses two integers $L$ and $R$ ($1 \le L \le R \le N$), then the device will tell the number of different materials of balls $L$, $L + 1$, $\ldots$, $R$.

After using the device, Pak Dengklek will colour each ball with one of the available $N$ colours, which are one of colour $1$, colour $2$, $\ldots$, colour $N$. Pak Dengklek asks for your help to use the device then colour the balls. If there are more than one possible colouring, you may colour them using any colouring.

Constraints

  • $2 \le N \le 1000$

Subtasks

  1.   (8 points) $Q = 10\;000$, balls with the same material have consecutive numbers.
  2.   (7 points) $Q = 2000$, balls with the same material have consecutive numbers.
  3. (19 points) $Q = 2000$, there are at most $3$ different materials.
  4. (14 points) $Q = 2000$, there are at most $4$ different materials.
  5. (12 points) $Q = 2000$, there are at least $N - 1$ different materials.
  6. (21 points) $Q = 10\;000$, $N \le 100$
  7. (19 points) $Q = 10\;000$

Problem Type Information

This is an "interactive" problem. In this problem, you will interact with the judge's program through standard input (stdin) and standard output (stdout). See the following interaction carefully.

Interaction

In the beginning, you will be given an integer $T$ representing the subtask number, then you will be given $N$ and $Q$ with the following format: 

T
N Q

Then, for not more than $Q$ times, you can output to the standard output with the following format:

? L R

This means you give the values $L$ and $R$ to the device. The values of $L$ and $R$ must satisfy $1 \le L \le R \le N$. For each of the outputs, the number of different materials (denoted by $X$) will be given by the device from input with the following format: 

X

In the end, you have to answer by output with the following format: 

! A1 A2 â€¦ AN

This means you colour ball $1$, ball $2$, $\ldots$, ball $N$ with colours $A_1$, $A_2$, $\ldots$, $A_N$ respectively.

If you can colour the balls correctly without using the device more than $Q$ times, then you will receive Accepted. If you give the wrong colouring, use the device more than $Q$ times, or output with the wrong format, then you will receive Wrong Answer.

Warning

Always flush every time your program gives an output.

  • For library stdio.h (cstdio):
    fflush(stdout);
  • For library iostream:
    std::cout << std::flush;
