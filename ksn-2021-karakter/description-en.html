Description

Pak Dengklek has an independent word $S$ which consists of $N$ characters. A word is said to be independent if all its characters are independent. A character in a word is said to be independent if fulfils two conditions below:

  • different from the previous character or is the first character in that word,
  • different from the next character or is the last character in that word.

For an instance, ABAC is an independent word but AABC is not, because the first and the second character in AABC are not independent characters.

Apparently, Pak Dengklek is not satisfied with the word $S$. Pak Dengklek wants to make a word $T$ by scrambling the characters in $S$. Of course, $T$ must be an independent word.

Pak Dengklek prefers a lexicographically smaller word. Therefore, Pak Dengklek asks for your help to obtain the lexicographically smallest $T$. It is possible that there is no possible $T$ which is lexicographically smaller than $S$, for this case, you do not need to scramble the characters in $S$ so that $T = S$.

What is lexicography?
A word $S$ is said to be lexicographically smaller than word $T$ if $S$ appeared before $T$ in the dictionary. Formally, a word $S$ that consists of $N$ characters is said to be lexicographically smaller than word $T$ that also consists of $N$ characters if there is $i$ ($1 \le i \le N$) so that $S_1 = T_1$, $S_2 = T_2$, $\ldots$, $S_{i-1} = T_{i-1}$, and $S_i < T_i$.

Constraints

  • $1 \le N \le 100\;000$
  • $S$ only consists of uppercase letters.
  • It is guaranteed that $S$ is an independent word.

Subtasks

  1.   (8 points) Contains only the following test case:
    $N = 7$ and the independent word $S$ is ACBACAB.
  2.   (6 points) There is no possible $T$ which is lexicographically smaller than $S$.
  3. (11 points) $S$ only consists of characters A and B.
  4. (10 points) There is no character that appeared more than once in $S$.
  5. (14 points) $N \le 10$
  6. (13 points) $N \le 200$
  7. (15 points) $N \le 3000$
  8. (23 points) No additional constraints.

Input

The input is given with the following format: 

N
S

Output

Output a line that contains a word $T$.

Sample Input 1

5
BADBC

Sample Output 1

ABCBD

Sample Input 2

3
ABA

Sample Output 2

ABA

Explanation of Sample 2

There is no other independent word that can be obtained other than $S$ itself.
